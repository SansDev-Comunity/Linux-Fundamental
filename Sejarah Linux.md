# Sejarah Linux

Linux merupakan operation system open source yang telah mengubah wajah teknologi dunia. Dari mendominasi server dan superkomputer hingga menjadi inti inovasi modern.
Dikala itu sudah ada sistem operasi komersial yaitu Unix dan DoS yang sekarang menjadi windows, kedua sistem operasi tersebut bersifat tertutup dan harus bisa didapatkan dengan cara membeli.

Berbeda dengan Windows yang menggunakan tipe file FAT-32 dan NTFS, tipe file yang digunakan oleh sistem operasi ini secara umum adalah ext2, ext3, ext, atau btrfs. 

Berawal dari proyek kecil seorang mahasiswa Finlandia, Linux kini digunakan oleh jutaan orang di berbagai industri. Perjalanannya adalah kisah kolaborasi, kebebasan teknologi, dan pengaruh besar terhadap industri.

Sistem operasi Linux adalah salah satu inovasi teknologi terbesar dalam dunia komputasi. Linux dibuat oleh Linus Torvalds pada tahun 1991 yang awalnya sebagai proyek pribadi untuk menciptakan kernel open-source yang terinspirasi dari sistem operasi Unix. Dahulu, proyek ini dinamai Freax, menggabungkan kata “free,” “freak,” dan “x” untuk Unix. Namun, seorang administrator FTP bernama Ari Lemmke mengunggah proyek ini ke internet dengan nama Linux tanpa seizin Torvalds. Kemudian, nama ini diterima oleh komunitas global dan tetap digunakan hingga sekarang sebagai identitas sistem operasi.

Tahun 1992 ketika Torvalds memutuskan untuk merilisnya di bawah lisensi GNU General Public License (GPL). Langkah ini memungkinkan siapa saja untuk menggunakan, memodifikasi, dan mendistribusikan ulang kode sumber Linux secara bebas. Kombinasi antara kernel Linux dan perangkat lunak dari proyek GNU.

Seiring waktu, Linux mulai berkembang menjadi berbagai distribusi (distro) untuk memenuhi kebutuhan yang berbeda. Slackware, distro pertama, dirilis pada tahun 1993.
